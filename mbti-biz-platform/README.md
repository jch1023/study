# MBTI Biz Navigator (MVP)

사업가 성향(MBTI)에 따라 아래를 분석하는 웹 MVP입니다.

- 사업 방향 추천 (무료)
- 마케팅/직원 관리/자동화 전략 (유료 미리보기 + 잠금)
- 심층 보고서 형태로 확장 가능한 구조

## 제품 전략

### 1) 트래픽 확보용 무료 정책
- MBTI 기반 사업 방향 요약: 무료 제공
- 심층 보고서 본문: 1,000자까지만 무료 공개
- 나머지: 구독 또는 건별 결제

### 2) 수익화(스이카) 구조
- Starter(월 구독): 심층 보고서 3회/월
- Pro(월 구독): 무제한 보고서 + 자동화 템플릿
- One-time 리포트 구매: 1회 결제

### 3) 확장 아키텍처 제안
- Frontend: 정적 웹(현재) → Next.js 전환 가능
- Backend: Python(FastAPI) 또는 Node(Nest)
- AI 분석: 프롬프트 템플릿 + MBTI 지식 베이스 + 사용자 입력
- 결제: Stripe/토스페이먼츠
- 리포트: HTML → PDF 렌더링 + 차트 이미지 삽입

## 실행 방법

```bash
cd mbti-biz-platform
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 테스트 시나리오 예시
1. MBTI: `ENTJ`
2. 관심 업종: `교육 SaaS`
3. 목표: `6개월 내 월 매출 3000만원`
4. 무료 섹션에서 사업 방향/추천 업종/성향 그래프 확인
5. 유료 섹션에서 1,000자 미리보기 + 잠금 메시지 확인
